# Sai Charan Portfolio

Personal portfolio website built with Next.js to showcase my full-stack and AI/ML projects.

## Tech Stack

- Next.js
- React
- CSS
- Resend

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Contact Form

Add these values in `.env.local`:

```env
RESEND_API_KEY=your_resend_api_key
CONTACT_TO_EMAIL=your_email_address
CONTACT_FROM_EMAIL=Portfolio Contact <onboarding@resend.dev>
```

For production, use a verified domain email in `CONTACT_FROM_EMAIL`.
